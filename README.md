## 프로젝트 소개

본 프로젝트는 **React와 TypeScript를 기반으로 구현한 날씨 정보 조회 웹 애플리케이션**입니다.  
사용자의 현재 위치를 기반으로 날씨 정보를 제공하며, 원하는 지역을 검색하여 해당 지역의 날씨를 확인할 수 있습니다.

또한 대한민국 행정구역 데이터를 활용하여 시, 구, 동 단위의 유연한 검색 기능을 제공하고,  
자주 확인하는 지역은 즐겨찾기로 등록하여 빠르게 접근할 수 있도록 구현했습니다.

사용자 경험을 고려하여 **반응형 UI**로 구성되었으며, 직관적인 인터페이스를 통해  
현재 날씨와 주요 기온 정보를 쉽게 확인할 수 있도록 설계되었습니다.

이를 통해 사용자 중심의 날씨 정보 탐색 경험을 제공하는 것을 목표로 합니다.

---

## 구현 기능

### 날씨 정보 조회
- 현재 기온
- 최저 / 최고 기온
- 외부 API를 활용한 실시간 데이터 조회

### 위치 기반 서비스
- 현재 위치 자동 감지
- Geolocation API 활용

### 장소 검색
- “서울”, “종로구” 등 검색 가능  
- 행정구역 JSON 데이터를 기반으로 자동완성 제공

### 즐겨찾기
- 최대 6개 저장 가능
- 카드 형태 UI로 빠른 접근 지원

## 기술 스택

### Frontend
- **React**
  - Functional Component 기반으로 개발
- **TypeScript**
  - 정적 타입을 활용한 안정적인 코드 작성

### Architecture
- **FSD (Feature-Sliced Design)**
  - 기능 단위로 구조를 분리하여 유지보수성과 확장성 향상

### Data Fetching
- **TanStack Query**
  - 서버 상태 캐싱 및 비동기 데이터 관리 효율화

### API
- 공공데이터포털 / OpenWeatherMap 등 외부 날씨 API 활용

### Data
- 대한민국 행정구역 JSON 데이터 활용
  - 시/구/동 단위 검색 기능 구현

### UI / UX
- **Tailwind CSS**
  - 유틸리티 기반 스타일링
- 반응형 웹 지원 (Desktop / Mobile)
